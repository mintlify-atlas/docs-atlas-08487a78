## What Makes JCV Fitness Different

Unlike generic fitness apps, JCV Fitness creates truly personalized programs based on your unique profile:

- **9-Step Interactive Wizard** - Answer questions about your experience, goals, time availability, equipment, and preferences
- **Personalized Meal Plans** - Get customized nutrition plans with food exchange tables and portion guidance
- **Flexible Workout Programs** - Choose between gym or home workouts based on your equipment and location
- **100+ Exercise Library** - Comprehensive database with proper form guidance and video tutorials
- **Protected PDF Downloads** - Download your plans with personalized watermarks for offline access
- **Multi-Tier Subscriptions** - Flexible pricing plans from Basic to Premium with MercadoPago integration

## How It Works

## Technology Stack

Built with modern, scalable technology:

- **Frontend**: Next.js 16, React 19, TypeScript, Tailwind CSS 4
- **State Management**: Zustand 5 for efficient client-side state
- **Backend**: Supabase for authentication, database, and row-level security
- **Payments**: MercadoPago integration with webhook handling via Cloudflare Workers
- **Hosting**: Cloudflare Pages with static export for optimal performance
- **Database**: PostgreSQL with row-level security policies protecting user data
